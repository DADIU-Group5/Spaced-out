Floors are 4x4 unity units.

doors are 3.3 unity units thick. so if you have a floor on X:0 and have a door, then the next floor would be at X:7.3 (4+3.3)

outer cornors should be placed on the tile it 'overlaps'
inner cornors should be placed on the 'empty' tile it 'overlaps'

between wall decor can be a little hard to place. Should be placed on a tile and rotated around.
